🏥 SaudINOB — Gestão de Doenças RespiratóriasBem-vindo ao SaudINOB, um ecossistema Full-Stack (Node.js + SQLite) desenhado para a monitorização inteligente de doentes com Asma e Rinite Alérgica.Este projeto vai além de um simples CRUD; ele implementa lógica clínica real através do questionário CARAT (Control of Allergic Rhinitis and Asthma Test) e um motor de validação baseado em JSON Schemas para garantir interoperabilidade e segurança.✨ Funcionalidades de Elite📊 Dashboard de Controlo: Visualização em tempo real de utentes, médicos e histórico clínico.🩺 Questionário CARAT Digital: Interface interativa para submissão de avaliações com cálculo automático de score.🚨 Motor de Alertas Inteligente: * Gera alertas automáticos se o score for inferior a 24.Deteta quedas de performance clínica (deterioração de $\ge 4$ pontos face à última consulta).🛡️ Validação por Contrato (Interoperabilidade): Utilização de AJV e JSON Schemas para garantir que os dados trocados entre sistemas cumprem padrões rigorosos.🧹 Arquitetura Limpa: Separação clara entre Rotas, Controladores, Middlewares e Lógica de Negócio (MVC).🛠️ Tecnologias e FerramentasCamadaTecnologiaUtilizaçãoBackendNode.js / ExpressMotor principal da APIBase de DadosSQLite3Persistência de dados relacional com Foreign KeysValidaçãoAJV / JSON SchemaValidação estrutural e contratos de dadosSegurançaexpress-validatorSanitização de inputs e proteção contra XSSFrontendVanilla JS / CSS GridInterface responsiva e dinâmica🚀 Como Instalar e Configurar1.
